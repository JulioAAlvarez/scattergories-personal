## Setup

### Prerequisites
- Node.js
- PostgreSQL
- Yarn (enable with `corepack enable`)

### Installation

```bash
# Clone the repo
git clone git@github.com:ABushUnderTheTree/scattergories_project.git
cd scattergories_project

# Install dependencies
yarn install

# Create .env file with your database credentials
# DATABASE_URL="postgresql://postgres:password@localhost:5432/scattergories_db"

# Create the database
createdb scattergories_db
```
